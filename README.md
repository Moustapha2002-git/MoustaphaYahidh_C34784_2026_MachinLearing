# Mini-Projet ML 2026 — Apprentissage Supervisé Linéaire (M1 IA)

Ce projet consolide les bases de l’apprentissage supervisé à travers deux modèles fondamentaux :
- **Régression Linéaire** (prédiction d’une variable continue)
- **Régression Logistique** (classification binaire sur Iris)

##  Objectifs
### Partie 1 — Régression Linéaire
- Choisir un dataset de régression (ex: *Medical Insurance Cost* / *House Prices*)
- Analyser les corrélations via **Heatmap**
- Formaliser le modèle : **y = β0 + Σ βi xi + ε**
- Évaluer le modèle avec **R²** et **MSE**
- Interpréter l’importance des variables via les coefficients **βi**

### Partie 2 — Régression Logistique
- Utiliser **Iris** (scikit-learn) et transformer en **binaire** (ex: classe 0 vs autres)
- Normaliser les variables d’entrée (**StandardScaler**)
- Modéliser la probabilité via la **sigmoïde**
- Évaluer via **Matrice de confusion**
- Calculer **Accuracy**, **Precision**, **Recall**
